# Modular Calculator (Java)

## Overview
This project is a **Java-based calculator application** designed to demonstrate:
- Method design and modularity
- Method overloading
- Utility methods for reusability
- Pass-by-value behavior in Java
- Exception handling (divide-by-zero)
- Clean structure to reduce redundancy
- Independent testing of each method

The project is ideal for interview preparation and learning core Java concepts.

---

##  Tools
- **Primary IDE:** IntelliJ IDEA  
- **Alternative IDE:** Eclipse  

---
